Structured Journalism

Developers at Nerve Centre Garage

Nerve Centre garage is a team of dedicated developers under DXC technology's AI Centre of Excellence.

This team comprises of various front-end and back-end developers, AI scientists , designers and evangelist and who are working on this awesome domain of structured journalism! This page describes more about our sub-teams within Nerve Centre Garage. If you would like to check out each team's repositories, we have provided the links below. If you would like to check our project Github , you can find it here.
Nerve Centre Garage has four sub-teams each working on different component of this whole ecosystem of Structured Journalism.
  • Team Article Production

    • This team forms the baseline foundation of data in this project. Using Python and a Flask application , they are creating a refined MongoDB database by dividing articles into snippets, on which sentiment, summary and similarities are measured using sentiment analysis, summary generator and similarity checker. These snippets are further clustered into topics and topics related to a theme are fetched by journalist using elastic search. Thus, it provides some to support journalists on consolidating the final stories and to find any tedium contents in the final content prepared for publishing before the approval process. Click here to know more about article production team. The entire system is set on docker and deployment is done using Heroku.

  • Team Journalism Automation

    • This team is working on Node-red, a flow based tool that provides some great automation to everyday work of Journalists and Editors. The team is focusing on automating most of the steps carried out within the Journalism ecosystem using finite state machines to make the process simpler in assigning stories based on the journalist's availability, notifying the editors on stories completed for them to review and approve and then send it forward to publish it . Click here to find Automation Team Repository

  • Content Templates team
    • Is a Group of Front-end developers focusing on creating responsive and more persona based templates to publish the content(stories) approved by the Editor , on a mini-site for audiences. To know more about Content template click here
